# SQL Sales Analysis – Tech Merchandise Store

## Project Overview
This project analyzes sales data for a coder merchandise store selling products such as hoodies, mugs, stickers, notebooks, and accessories.

The goal of the project is to design a relational database and perform business analysis using SQL queries.

## Database Structure
The database contains the following tables:

- customers
- products
- orders
- order_items
- payments

## Key Analysis Queries

The project answers important business questions such as:

• What is the total revenue generated?  
• What is the average order value?  
• Which products generate the highest revenue?  
• Who are the top spending customers?  
• Which products are the best selling?  
• How many orders were cancelled?  
• Which customers spent more than the average customer?
